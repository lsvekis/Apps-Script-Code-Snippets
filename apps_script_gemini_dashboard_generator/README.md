# AI Spreadsheet Dashboard Generator (Apps Script + Gemini)

Generate dashboard sheets in Google Sheets from plain English using Apps Script + Gemini.

## Setup
1. Open a Google Sheet
2. Extensions → Apps Script
3. Replace files with this repo
4. Set `GEMINI_API_KEY` in `src/GeminiHelpers.gs`
5. Run `testGeminiConnection()`
6. Reload the sheet and use **AI Tools → AI Dashboard Generator**
