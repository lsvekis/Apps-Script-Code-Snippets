# Google Apps Script — 100 More Examples (101–200)

This repository contains 100 additional examples, numbered **101–200**, each in its own folder.

- `examples/README.md` contains the full index.
- Each example folder contains:
  - `README.md` (what it does + notes)
  - `Code.gs` (copy/paste runnable code)

> Some examples require replacing placeholders like `PASTE_FOLDER_ID` / `PASTE_FILE_ID` and authorizing services.
